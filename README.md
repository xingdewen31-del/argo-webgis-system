# Argo WebGIS Visualization System

## Introduction
This project is a WebGIS-based Argo buoy visualization system.  
It integrates Vue, Cesium, FastAPI and PostgreSQL/PostGIS to achieve spatial data visualization and query.

## Tech Stack
- Vue
- Cesium
- FastAPI
- PostgreSQL/PostGIS

## Features
- 3D globe visualization
- Argo buoy spatial distribution display
- Buoy information popup
- Temperature and salinity profile display
- Basemap switching
- Multi-condition buoy query

## Run Project

Install dependencies:

npm install

Start project:

npm run dev
